# Financial Management System (FMS)

Professional, student‑friendly Financial Management System with:
- Database migrations and seeds (MySQL 8) in /db
- OpenAPI 3.0 API contract in /docs
- Postman collection in /postman
- React + Vite frontend (JavaScript only) in /client
- No backend/server code included

The frontend consumes any future backend via the environment variable VITE_API_BASE and is resilient when no backend is present.
